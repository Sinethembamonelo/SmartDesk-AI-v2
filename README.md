# SmartDesk-AI-v2
SmartDesk-AI V2 is a beginner-friendly AI productivity assistant designed to help users organized and improved focus using a clean web interface.
## Purpose
This project was created to explore how AI-inspired productivity tools can be built using basic fronted web development technologies.
## Features
- Clean and simple user interface
- Interactive "Get Started" button
- Beginner-friendly project structure
- Lightweight and responsive design
## Built with
- HTML
- CSS3
- JavaScript
## Goals
- Learn web development step by step
- Practice building AI-inspired applications
- Improve coding and problem-solving skills
- Understand how fronted projects are structured
## Future Improvements
- AI chatbot assistant
- Task management system
- dark mode support
- Voice assistant features
- Productivity tracking tools
## Project Structure
- 'index.html' (Main webpage structure)
-  'style.css' (Styling and layout)
- 'script.js (Interactivity and functionality)
